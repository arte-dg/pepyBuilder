== Description ==
HT Builder is a drag & drop theme builder plugin for Elementor Page Builder. Theme Builder is included in this plugin to build custom blog page, custom blog details page, search page, archive page, coming soon page, 404 page and custom template for header/footer.

20 Element Addons, awesome styling options. You can easily build your blog page, blog details page, search page, 404 page, unlimited header, unlimited footer layout from the elementor page builder.

* [Demo](http://demo.wphash.com/2/ht-builder/)
* [Support!](https://hasthemes.com/contact-us/)

== Installation ==
This section describes how to install the HT Builder For Elementor Page Builder Plugins for WordPress get it working.

= 1) Install =

= Install: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "HT Builder".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `ht-builder` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
